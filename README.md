UrbanStyle Customer Data Cleaning:
Overview:
This project focuses on cleaning and preparing messy customer data for analysis using Python and Pandas. The dataset includes customer demographics, purchase history, and loyalty information, with intentional data quality issues that must be resolved.

Objectives:
Load and explore raw customer data
Handle missing values and inconsistencies
Standardize text formats (names, categories, phone numbers)
Convert and fix incorrect data types
Remove duplicate records
Create new useful features for analysis
Generate business insights from cleaned data

Tools Used:
Python
Pandas
NumPy
Regular Expressions (re module)
Datetime handling

Data Cleaning Steps:
Filled missing values using median and forward-fill strategies
Standardized inconsistent text formats (names, categories, phone numbers)
Converted currency and numeric fields to correct data types
Removed duplicate customer records
Created derived features such as:
Days since last purchase
Average purchase value
Purchase frequency category

Key Insights:
Loyalty status impacts average customer spending
Electronics/Footwear-related customers tend to spend more
Data inconsistencies (dates, phone formats, missing values) were common
After cleaning, dataset is ready for reliable business analysis
Customer segmentation becomes more accurate after standardization

What I Learned:
How to clean real-world messy datasets
Handling missing data using different strategies
Standardizing inconsistent formats across columns
Removing duplicates while preserving meaningful records
Creating derived features for business insights
Preparing raw data for analytics and reporting

Summary:
This project demonstrates how raw, inconsistent customer data can be transformed into a clean, structured dataset ready for business analysis and decision-making.
